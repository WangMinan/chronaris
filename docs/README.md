## 目录说明

`docs` 目录按二级分类组织为五组：

1. `foundation`：项目边界、分层、架构与基础契约
2. `planning`：路线图、阶段执行计划与实现记录
3. `models`：模型输入协议、阶段性模型设计与参考资料
4. `reports`：单架次 preview、验证摘要与阶段性核验结果
5. `选题报告与基金申请书`：选题与基金原始文档
