# Docs Index

## 1. 目录说明

`docs` 目录按二级分类整理为四组：

1. `foundation`
   记录项目边界、分层、架构与基础契约
2. `planning`
   记录路线图、阶段总结与推进状态
3. `models`
   记录模型输入协议、阶段性模型设计与参考资料
4. `reports`
   记录单架次 preview、验证摘要与阶段性核验结果
